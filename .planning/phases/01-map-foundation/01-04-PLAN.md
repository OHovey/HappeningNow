Create `src/lib/map/layers.ts`:
- Export layer definition objects (not functions that call map.addLayer — just the config objects):

`eventCircleLayer`: Circle layer for individual event markers
- `id: 'event-circles'`
- `type: 'circle'`
- `source: 'events'`
- `filter: ['!', ['has', 'point_count']]` (not clusters)
- Paint: circle-color uses `['match', ['get', 'category'], 'festival', '#f97316', 'wildlife', '#22c55e', '#3b82f6']`
- circle-radius interpolated by scale: `['interpolate', ['linear'], ['get', 'scale'], 1, 5, 10, 12]`
- circle-stroke-width: 1, circle-stroke-color: '#ffffff'

`pulseLayer`: Overlay circle for pulse animation
- `id: 'event-circles-pulse'`
- Same source, same filter as eventCircleLayer
- Paint: same color match, circle-radius: 8, circle-opacity: 0.4

`clusterLayer`: Circle layer for clusters
- `id: 'clusters'`
- `filter: ['has', 'point_count']`
- Paint: circle-color stepped by point_count: `['step', ['get', 'point_count'], '#51bbd6', 10, '#f1f075', 50, '#f28cb1']`
- circle-radius stepped: `['step', ['get', 'point_count'], 15, 10, 20, 50, 25]`

`clusterCountLayer`: Symbol layer for cluster numbers
- `id: 'cluster-count'`
- `filter: ['has', 'point_count']`
- Layout: text-field: `'{point_count_abbreviated}'`, text-font: `['Noto Sans Regular']`, text-size: 12

Create `src/lib/map/animations.ts`:
- Export `startPulseAnimation(map: maplibregl.Map): () => void`
- Uses requestAnimationFrame loop
- Modulates pulse layer opacity (0.1 to 0.4) and radius (8 to 14) using `Math.sin(timestamp / 500)`
- Returns a cleanup function that cancels the animation frame

Update `src/components/map/MapView.tsx` in the `map.on('load')` handler:
- Add all 4 layers in order: clusterLayer, clusterCountLayer, eventCircleLayer, pulseLayer
- Start pulse animation, store cleanup ref
- Add cluster click handler: query features, get cluster expansion zoom, easeTo
- Add NavigationControl (top-right)
- Add GeolocateControl (top-right) with `positionOptions: { enableHighAccuracy: true }`, `trackUserLocation: true`
- Add FullscreenControl (top-right)
- Change cursor to pointer on hover over event-circles and clusters
- On unmount: call pulse animation cleanup + map.remove()

Include sample hardcoded GeoJSON (5-10 events across different continents) so the map has visible markers even before seed data is loaded. This gets replaced by Supabase fetch in plan 01-05.
