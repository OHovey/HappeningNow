Create `src/lib/affiliates.ts`:
- Export `buildBookingLink(params: { destinationId?: string; city?: string; checkin?: string; checkout?: string }): string`
  - Base URL: `https://www.booking.com/searchresults.html`
  - Params: `aid` from `NEXT_PUBLIC_BOOKING_AFFILIATE_ID`, `dest_id`, `dest_type: 'city'`, `ss` (city fallback), `checkin`, `checkout`, `no_rooms: '1'`, `group_adults: '2'`
  - If no destinationId, use city name as `ss` search param
  - If no checkin/checkout, derive from event months (start of start_month, end of start_month)
  - Return full URL string
- Export `buildGetYourGuideLink(params: { locationId?: string; query?: string }): string`
  - If locationId: `https://www.getyourguide.com/l${locationId}`
  - Else: `https://www.getyourguide.com/s/`
  - Append: `partner_id` from `NEXT_PUBLIC_GYG_PARTNER_ID`, `q` from query
  - Return full URL string
- Both functions should handle missing env vars gracefully (return "#" or a fallback URL, don't crash)
