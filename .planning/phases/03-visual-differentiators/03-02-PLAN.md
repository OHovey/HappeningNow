2. Create `src/lib/map/migration-layers.ts` with:
   - `SPECIES_COLORS: Record<string, string>` — distinct colors per species group: whales='#0ea5e9' (sky blue), wildebeest='#a3660a' (brown), butterflies='#f59e0b' (amber), birds='#8b5cf6' (violet), caribou='#10b981' (emerald), turtles='#06b6d4' (cyan), default='#6b7280' (gray). Per user decision: all species visible simultaneously with distinct colors.
   - `getSpeciesColor(species: string): string` — lowercase lookup in SPECIES_COLORS with gray fallback.
   - `splitRouteAtPosition(coordinates: number[][], positionIndex: number): { completed: number[][], upcoming: number[][] }` — splits the route coordinate array at the given index to create two LineStrings for the trail effect (solid behind dot, dashed ahead). Per user decision: trail effect gives a sense of direction and progress.
   - `buildRouteSources(route: MigrationRouteWithGeoJSON, selectedMonth: number)` — returns `{ completedSource, upcomingSource, dotSource }` as three GeoJSON objects:
     - Compute position index using `computeActivePosition` with `selectedMonth`.
     - Split coordinates using `splitRouteAtPosition`.
     - `completedSource`: LineString from start to position.
     - `upcomingSource`: LineString from position to end.
     - `dotSource`: Point at the computed position.
   - `createRouteLayerPair(routeId: string, species: string): { completedLayer, upcomingLayer }` — returns two LayerSpecification objects:
     - Completed layer: `id: 'route-{routeId}-completed'`, type: 'line', solid, `line-width: 3`, `line-opacity: 0.9`, color from `getSpeciesColor(species)`, `line-cap: 'round'`, `line-join: 'round'`.
     - Upcoming layer: `id: 'route-{routeId}-upcoming'`, type: 'line', `line-dasharray: [2, 4]`, `line-width: 2`, `line-opacity: 0.4`, same color.
   - `createDotLayer(routeId: string, species: string): LayerSpecification` — circle layer, `id: 'route-{routeId}-dot'`, `circle-radius: 7`, `circle-color` from species, `circle-stroke-width: 2`, `circle-stroke-color: '#ffffff'`, `circle-opacity: 1`. Per user decision: pulsing dot indicator, universal across all species.
   - `createDotPulseLayer(routeId: string, species: string): LayerSpecification` — overlay circle for pulse animation, `id: 'route-{routeId}-dot-pulse'`, same source, `circle-radius: 12`, `circle-opacity: 0.3`, same color.
