    **src/app/robots.ts:**
    Use Next.js built-in robots.ts file convention per RESEARCH.md Pattern 5:
    ```typescript
    import type { MetadataRoute } from 'next';

    export default function robots(): MetadataRoute.Robots {
      return {
        rules: {
          userAgent: '*',
          allow: '/',
          disallow: ['/api/'],
        },
        sitemap: 'https://happeningnow.travel/sitemap.xml',
      };
    }
    ```

    This:
    - Allows all crawlers to access all pages
    - Blocks /api/ routes (internal API endpoints)
    - Points to the sitemap index at /sitemap.xml
    - Uses Next.js file convention for proper caching and content-type
