    Export `shouldNoindex(eventCount: number, hasWeatherData: boolean, hasCrowdData: boolean): boolean`.

    Export functions that generate related page links for cross-linking:

    - `getRelatedWildlifeLinks(currentRegion, currentSpecies, currentMonth)` — returns links to same-region different species, same-species different regions
    - `getRelatedWhatToDoLinks(currentDestination, currentMonth, allDestinations)` — returns links to same-destination adjacent months, nearby destinations same month
    Each function returns `Array<{ href: string; label: string }>` with max 8 links.

    - `getWildlifeByRegionMonth(region, month)` — wildlife events by region and month
    - `getWhatToDoData(destinationSlug, month)` — events + destination crowd/weather for a destination in a month

    - `getAllEventSlugs()` — returns all event slugs for root sitemap (used by Plan 04-05 sitemap.ts)
    - `getAllWildlifeSlugs()` — returns all wildlife/migration route slugs for root sitemap (used by Plan 04-05 sitemap.ts)
