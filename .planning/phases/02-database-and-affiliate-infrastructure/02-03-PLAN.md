1. Create `src/lib/supabase/queries.ts` with server-side data helpers:
   - `getEventBySlug(slug: string)`: Fetches single event with all fields from events table. Returns `Event | null`.
   - `getAllEventSlugs()`: Fetches `select('slug')` from events table. Returns `string[]`. Used by generateStaticParams.
   - `getNearbyEvents(event: Event, limit?: number)`: Fetches events in the same region OR overlapping month range, excluding the current event, limited to 4-6 results. Use geographic proximity (same region) as primary criterion, temporal overlap as secondary. Returns `Event[]`.
   - `getWildlifeBySlug(slug: string)`: Fetches single migration_route by slug. Returns `MigrationRoute | null`. (Shared helper for plan 02-04.)
   - `getAllWildlifeSlugs()`: Fetches slugs from migration_routes. Returns `string[]`. (Shared helper for plan 02-04.)
   - All functions use `createServerClient()` from `@/lib/supabase/server`.

2. Create `src/components/detail/EventHero.tsx`:
   - Full-width hero image (object-cover, aspect-ratio ~3:1 or 2.5:1) with event photo from `image_url`
   - If no image: render a gradient background using category color from constants (orange for festivals, green for wildlife)
   - Overlay: event name (large white text with text-shadow) and date range (formatted using `formatMonthRange`)
   - CrowdBadge positioned in top-right corner of hero

3. Create `src/components/detail/EventContent.tsx`:
   - Renders below the hero in a max-w container
   - Sections: description text, dates + location info, mini map (import from 02-02), affiliate CTAs with FTC disclosure
   - Import and render `AffiliateLinks` from `src/components/panel/AffiliateLinks.tsx` (existing from Phase 1)
   - Import and render `FtcDisclosure` below affiliate links
   - Import and render `MiniMap` with event coordinates (lazy-loaded via next/dynamic ssr:false)
   - Pass event location coordinates extracted from the event's geometry data (need to query with `ST_X(location)` and `ST_Y(location)` or use a separate select)

4. Create `src/components/detail/NearbyEvents.tsx`:
   - Props: `events: Event[]`
   - Renders a grid of event cards (image thumbnail, name, date range, crowd badge)
   - Each card links to `/event/${event.slug}`
   - If no nearby events, render nothing (return null)

   - Test: description falls back to "Discover {name}" when event description is null
