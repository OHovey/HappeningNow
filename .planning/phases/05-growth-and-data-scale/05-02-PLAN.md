1. Create `src/app/search/page.tsx` — Server component shell:
   - Metadata: title "Search Events Near You | HappeningNow.travel", description about finding events by location and dates
   - Renders a client-side SearchPage component (use next/dynamic with ssr:false for map, or a 'use client' wrapper)
   - The SearchPage manages state: selectedLocation ({lat, lng, name} | null), dateRange ({start: number, end: number}), category (string | null), radius (number, default 200000), results (array), selectedResultId (string | null)
   - URL state sync: all filters stored in searchParams (lat, lng, from, to, cat, radius) so search is bookmarkable
   - Layout: SearchBar at top, below it a flex row — SearchResults on left (w-full lg:w-1/2), SearchMap on right (hidden on mobile, lg:w-1/2 lg:block)
   - Auto-search: useEffect triggers fetch to /api/search when lat+lng are set, debounced 300ms on filter changes
   - Results scored client-side with worthTheTripScore, sorted descending

2. Create `src/components/search/SearchBar.tsx` — Horizontal form bar:
   - Google Flights style: single horizontal row with fields inline
   - Fields: LocationInput (takes full width on mobile, flex-1 on desktop), date range (two month selects 1-12 for start/end), category (select: All/Festivals/Wildlife/Other), distance (select: 50km/100km/200km/500km)
   - Tailwind: bg-white rounded-xl shadow-lg p-4 flex flex-wrap gap-3 items-end
   - No search button — all changes propagate via onChange callbacks to parent
   - On mobile: fields stack vertically (flex-col on sm, flex-row on lg)

3. Create `src/components/search/LocationInput.tsx` — Photon autocomplete input:
   - Text input with debounced fetch to /api/geocode?q={input} (300ms debounce)
   - Dropdown shows Photon results as selectable items: "City, State, Country" format from properties
   - On select: calls onSelect({lat, lng, name}) callback, closes dropdown, sets input text to selected name
   - On blur: closes dropdown after short delay (150ms to allow click registration)
   - Keyboard: ArrowDown/ArrowUp to navigate, Enter to select
   - Shows a subtle location pin icon on the left of the input
   - Renders "Searching..." text while loading, "No results" when empty response
